Hwk 3 (1)

Prove

  1. If \(A ≈ B\) and \(A\) is finite then so is \(B\).
  2. If \(A ≈ B\) and \(A\) is countably infinite then so is \(B\).
  3. If \(A ≈ B\) and \(A\) is uncountable then so is \(B\).

See Note (f), p.1-10.

Hwk 3 (2)

Prove Theorem 1.13 (c): \(\text{PRED}(ℕ)\) is uncountable. Use a diagonalization method.

Do not use the fact that \(\text{PRED}(ℕ)\) is equinumerous with \(𝒫(ℕ)\).
